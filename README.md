# book_club

이 리포지토리는 읽은 책에 대한 서평을 작성하고 공유하는 리포지토리 입니다.

## 목적
---
읽은 책의 내용을 공유하고, 책을 읽지 않은 사람들에게 지식을 전달하기 위함입니다.

이 리포지토리에 있는 글을 읽고, 이 책을 구매할 마음이 생긴다면 더욱 의미있는 글이 될 수 있습니다.

## 현재까지 작성된 책 목록
---
* 프로그래머의 뇌
* 실용주의 프로그래머


## 앞으로 작성될 책 목록
---
* 객체지향의 사실과 오해
* 클린 코드
* 클린 아키텍처
* 도메인 주도 설계
* 테스트 주도 개발
* 어떻게 문제를 풀 것인가
